# TLS Auth Server

Небольшой учебный проект, написанный для изучения сетевого программирования на C++.

Проект состоит из TLS-сервера и консольного клиента. Сервер принимает защищённые соединения через OpenSSL, выполняет авторизацию по токену через SQLite и обрабатывает команды от клиента.

## Возможности

* TLS 1.3 соединение через OpenSSL
* Авторизация по токену
* Хранение токенов в SQLite
* Многопоточная обработка клиентов
* Ограничение количества подключений
* Таймауты сокетов
* Корректное завершение работы через SIGINT/SIGTERM
* RAII для управления ресурсами
* Отдельный TLS-клиент для тестирования сервера

## Используемые технологии

* C++17
* OpenSSL
* SQLite3
* POSIX sockets
* std::thread

## Сборка

### Linux

```bash
g++ server/main.cpp server/server.cpp -o server \
-lssl -lcrypto -lsqlite3 -pthread

g++ client/main.cpp client/connection.cpp -o client \
-lssl -lcrypto -pthread
```

## Запуск

Сервер:

```bash
./server
```

Клиент:

```bash
./client
```

После подключения клиент отправляет токен авторизации:

```text
AUTH:my_secret_token_123
```

Если токен присутствует в базе данных, соединение считается авторизованным.

## Пример команды

```text
time
```

Ответ сервера:

```text
18:42:13
```
